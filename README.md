# CustomMeteorites

**CustomMeteorites** — это плагин для сервера [Paper](https://docs.papermc.io/) (версия 1.21.1), созданный для добавления в мир Minecraft редких событий — падения метеоритов. Эти космические артефакты приносят с собой редкие ресурсы, сокровища и даже охранников, добавляя азарт и исследовательский дух в ванильное выживание.

## Особенности

*   **Разнообразные метеориты:** Несколько типов метеоритов с уникальными внешним видом, шансом появления и составом.
*   **Настройка конфигурации:** Гибкая настройка через `config.yml`, включая шансы, размеры, взрывы, частицы, сокровища и охранников.
*   **Сокровища в ванильном стиле:** Сундуки/бочки с редкими и полезными предметами из разных миров (Overworld, Nether, End), включая алмазную/незеритовую броню, инструменты, оружие и уникальные артефакты с зачарованиями, соответствующими ванильному выживанию.
*   **Охранники:** После падения метеорита может появиться моб-охранник, защищающий сокровище.
*   **Интеграция с WorldGuard:** Возможность исключить падение метеоритов в определённые зоны.
*   **Частицы и эффекты:** Визуальные эффекты падения метеорита.
*   **Очистка:** Возможность автоматического удаления блоков метеорита через заданное время.
*   **Команды:** Команды для администраторов (`/meteor force`, `/meteor reload`).
