# Функции

## Метеориты

*   **Случайное появление:** Метеориты падают в мир с заданным интервалом.
*   **Разные типы:** Несколько предустановленных типов метеоритов (например, Огненный, Ледяной), каждый со своими характеристиками.
*   **Слоистая структура:** Метеориты состоят из ядра, внутреннего и внешнего слоёв, каждый из которых может быть из разных блоков.
*   **Физика падения:** `FallingBlock` с настраиваемой скоростью и начальной горизонтальной скоростью.
*   **Взрывы и молния:** Настройка мощности взрывов и появления молнии при падении для каждого слоя.
*   **Частицы:** Визуальные эффекты, сопровождающие падение.
*   **Очистка:** Возможность автоматического удаления блоков метеорита через заданное время.

## Сокровища

*   **Сундуки/Бочки:** После падения может появиться контейнер с сокровищами.
*   **Разнообразный лут:** Включает редкие предметы, броню, инструменты, оружие, ресурсы и артефакты (например, `Elytra`, `Dragon Egg`, `Nether Star`) из разных миров.
*   **Ванильные зачарования:** Все зачарования соответствуют правилам ванильного выживания.
*   **Креативный лор:** Уникальные описания для каждого предмета.

## Охранники

*   **Случайное появление:** После падения может появиться моб-охранник.
*   **Настройка:** Возможность настроить тип моба, здоровье, урон, скорость, снаряжение, имя, сообщения и звуки.

## Интеграции

*   **WorldGuard:** Возможность исключить падение метеоритов в указанные регионы.
