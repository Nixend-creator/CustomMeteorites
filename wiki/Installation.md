# Установка

1.  Убедитесь, что у вас запущен сервер [Paper](https://papermc.io/downloads/paper) версии **1.21.1**.
2.  Скачайте последнюю версию JAR-файла `CustomMeteorites-<version>-SNAPSHOT.jar` из релизов (если они есть) или соберите его из исходного кода с помощью `mvn clean package`.
3.  Поместите JAR-файл в папку `plugins` вашего сервера.
4.  Запустите (или перезапустите) сервер.
5.  Плагин автоматически сгенерирует файл `config.yml` в папке `plugins/CustomMeteorites/`.
6.  Отредактируйте `config.yml` под свои нужды.
7.  Перезапустите сервер или используйте команду `/meteor reload`, чтобы загрузить изменения.
